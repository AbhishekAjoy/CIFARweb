Image Prediction using CIFAR-10




CIFAR-10 is a dataset that contains large number of images of 10 objects. These images are used to train machine learning models. This is a model trained using CIFAR-10 dataset. If given any images of the 10 objects for which the model is trained, it will be able to predict the object with an accuracy of 84%.
